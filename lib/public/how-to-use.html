What is it?

This is the realtime web application with which you can see all operation data of Yamanote line. You can search where trains are, whether trains are under operation or not and how the delay of specific train is. We are making an effort to visualize a lot of information as possible in real-time. Enjoy Yamate!

How to use it?

All you have to do is only seeing. Current available data are listed below.

  • Positions of all trains
  • Operation information of all trains
  • Delay information of all trains

Source data

All data which we are using are provided by YRP Ubiquitous Networking Lab
